### Create ICNS for Tahoe

<img src=".assets/right.png" width="49.375%"/><img src=".assets/1x1.png" width="1.25%"/><img src=".assets/wrong.png" width="49.375%"/>

Tahoe ICNS files need 10% padding, which many icons from [macOSicons](https://macosicons.com/) lack. This command generates a properly padded ICNS from a 1024Ã—1024 PNG file, ideally exported from the [Icon Composer](https://developer.apple.com/icon-composer/) application.

```shell
curl -sSL https://raw.githubusercontent.com/olankens/runmagic/HEAD/src/create-tahoe-icns.sh | bash
```
